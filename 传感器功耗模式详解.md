## 📊 功耗模式总览

传感器的功耗模式设计对于电池供电设备至关重要。合理的功耗管理可以显著延长设备续航时间。

### 典型功耗模式分类

| 模式 | 功耗 | 唤醒时间 | 适用场景 |
|------|------|---------|---------|
| **主动测量** | 高 (1-50mA) | 即时 | 连续数据采集 |
| **待机/睡眠** | 中 (10-500μA) | 快速 (ms 级) | 间歇测量 |
| **深度睡眠** | 低 (1-10μA) | 中等 (10-100ms) | 低功耗待机 |
| **关机/断电** | 极低 (<1μA) | 慢 (100ms+) | 长期闲置 |

## 🔋 常见传感器功耗模式详解

### 1. IMU (惯性测量单元)

#### BMI088 (6 轴 IMU)

| 模式 | 加速度计 | 陀螺仪 | 总功耗 | 说明 |
|------|---------|--------|--------|------|
| 主动测量 | 1.0mA | 5.5mA | 6.5mA | 全功能运行 |
| 待机模式 | 60μA | 1.2mA | 1.26mA | 陀螺仪保持 |
| 深度睡眠 | 2μA | 7μA | 9μA | 寄存器保持 |
| 关机 | <1μA | <1μA | <2μA | 断电模式 |

**功耗优化策略**:
- 降低 ODR (输出数据速率)
- 使用 FIFO 缓冲减少 MCU 唤醒
- 启用智能运动触发唤醒
- 关闭未使用的轴

#### BMI270 (可穿戴 IMU)

| 模式 | 功耗 | 说明 |
|------|------|------|
| 正常测量 | 350μA @100Hz | 标准模式 |
| 低功耗模式 | 140μA @25Hz | 降低 ODR |
| 睡眠模式 | 60μA | FIFO 启用 |
| 待机模式 | 2μA | 仅寄存器保持 |

**特色功能**:
- 内置活动识别 (步数/姿态)
- 硬件 FIFO (1024 帧)
- 智能中断唤醒

### 2. 气压传感器

#### LPS22HB (防水气压)

| 模式 | 功耗 | ODR | 说明 |
|------|------|-----|------|
| 高分辨率 | 600μA | 75Hz | 最高精度 |
| 正常模式 | 380μA | 25Hz | 平衡模式 |
| 低功耗 | 60μA | 10Hz | 电池供电 |
| 待机模式 | 1.8μA | - | 断电保持 |
| 关机 | <0.5μA | - | 完全断电 |

**功耗优化策略**:
- 使用 FIFO 降低 MCU 负载
- 降低 ODR 到 1Hz (海拔监测足够)
- 启用阈值中断唤醒
- 使用单次测量模式

#### BMP280 (经典气压)

| 模式 | 功耗 | 说明 |
|------|------|------|
| 强制模式 | 2.7μA/次 | 单次测量，最省电 |
| 睡眠模式 | 0.1μA | 待机 |
| 正常模式 | 630μA | 连续测量 |

### 3. 气体传感器

#### SGP40 (VOC 气体)

| 模式 | 功耗 | 说明 |
|------|------|------|
| 测量模式 | 3.8mA | VOC 测量中 |
| 待机模式 | <1μA | 测量间隔 |
| 老化模式 | 3.8mA | 初始 168 小时 |

**功耗优化策略**:
- 降低测量频率 (1 次/分钟足够)
- 测量间隙进入待机
- 完成初始老化后功耗更低

#### SGP30 (前代 VOC)

| 模式 | 功耗 | 说明 |
|------|------|------|
| 测量模式 | 48mA (峰值) | 加热期间 |
| 待机模式 | <1μA | 测量间隔 |
| 睡眠模式 | 0.7μA | 深度睡眠 |

**注意**: SGP40 比 SGP30 功耗降低 90%+

### 4. ToF 测距传感器

#### VL53L1X (4m ToF)

| 模式 | 功耗 | 测量时间 | 说明 |
|------|------|---------|------|
| 连续测量 | 20mA | 持续 | 实时测距 |
| 单次测量 | 20mA | 30ms | 按需测量 |
| 低功耗自动 | 20mA (脉冲) | 30ms | 自动休眠 |
| 待机模式 | <1μA | - | 寄存器保持 |
| 关机模式 | <0.1μA | - | 断电 |

**功耗优化策略**:
- 使用单次测量模式
- 降低测量频率
- 缩短测量时间预算 (15-33ms)
- 启用距离阈值中断

#### VL53L0X (2m ToF)

| 模式 | 功耗 | 说明 |
|------|------|------|
| 连续测量 | 26mA | 持续运行 |
| 单次测量 | 26mA | 按需 |
| 睡眠模式 | <1μA | 待机 |

### 5. 温湿度传感器

#### SHT40 (高精度)

| 模式 | 功耗 | 测量时间 | 说明 |
|------|------|---------|------|
| 高精度测量 | 1.2mA | 8.2ms | ±0.1°C |
| 中精度测量 | 0.8mA | 4.5ms | ±0.2°C |
| 低功耗测量 | 0.5mA | 2.5ms | ±0.3°C |
| 待机模式 | <0.15μA | - | 完全休眠 |

**功耗优化策略**:
- 选择合适精度模式
- 降低测量频率
- 使用单次测量 (无加热)
- 避免频繁测量导致自加热

#### AHT20 (低成本)

| 模式 | 功耗 | 说明 |
|------|------|------|
| 测量模式 | 0.8mA | 正常测量 |
| 待机模式 | <50μA | 低功耗待机 |

## 📈 功耗计算实例

### 实例 1: 便携式空气质量监测仪

**配置**:
- SGP40 (VOC): 1 次/分钟
- SHT40 (温湿度): 1 次/分钟
- LPS22HB (气压): 1 次/5 分钟
- MCU: nRF52832 (BLE)

**功耗分析**:

| 组件 | 测量功耗 | 测量时间 | 频率 | 平均功耗 |
|------|---------|---------|------|---------|
| SGP40 | 3.8mA | 30ms | 1/min | 0.19μA |
| SHT40 | 1.2mA | 8ms | 1/min | 0.16μA |
| LPS22HB | 0.6mA | 10ms | 1/5min | 0.02μA |
| MCU 待机 | 1μA | 持续 | - | 1μA |
| BLE 广播 | 15mA | 1ms | 1s | 15μA |
| **总计** | | | | **~16.4μA** |

**电池寿命** (CR2032, 220mAh):
```
220mAh / 0.0164mA ≈ 13,400 小时 ≈ 1.5 年
```

### 实例 2: 无人机飞控传感器

**配置**:
- BMI088 (IMU): 200Hz 连续
- LPS22HB (气压): 50Hz 连续
- VL53L1X (避障): 10Hz

**功耗分析**:

| 组件 | 功耗 | 模式 |
|------|------|------|
| BMI088 | 6.5mA | 连续 200Hz |
| LPS22HB | 0.6mA | 连续 50Hz |
| VL53L1X | 20mA × 10% | 10% 占空比 |
| **总计** | **9.1mA** | |

**优化后** (使用 FIFO + 智能唤醒):
- BMI088: 3mA (FIFO + 降低 ODR)
- LPS22HB: 0.3mA (降低 ODR)
- VL53L1X: 2mA (按需测量)
- **优化后总计**: 5.3mA (节省 42%)

## 🔧 低功耗设计最佳实践

### 1. 传感器选择

| 优先级 | 考虑因素 |
|--------|---------|
| 1 | 待机电流 (<5μA 为佳) |
| 2 | 测量功耗/时间比 |
| 3 | 内置 FIFO/缓冲 |
| 4 | 智能中断功能 |
| 5 | 快速启动时间 |

### 2. 测量策略

```
推荐策略:
┌─────────────────────────────────┐
│ 1. 使用单次测量模式              │
│ 2. 降低测量频率到最低可接受值    │
│ 3. 启用 FIFO 减少 MCU 唤醒       │
│ 4. 配置阈值中断 (仅异常时唤醒)   │
│ 5. 测量间隙进入深度睡眠          │
└─────────────────────────────────┘
```

### 3. 硬件设计

- **电源域隔离**: 传感器独立 LDO，可单独断电
- **电源开关**: MOSFET 控制传感器电源
- **上拉电阻**: I2C 上拉电阻值优化 (4.7k-10k)
- **去耦电容**: 靠近传感器 VDD 引脚 (100nF)

### 4. 软件优化

```c
// 推荐测量流程
void sensor_measure_low_power(void)
{
    // 1. 唤醒传感器
    sensor_wakeup();
    
    // 2. 启动单次测量
    sensor_start_single();
    
    // 3. 等待数据就绪 (中断或轮询)
    wait_for_data_ready();
    
    // 4. 读取数据
    sensor_read_data();
    
    // 5. 立即进入待机
    sensor_enter_standby();
    
    // 6. MCU 进入睡眠
    mcu_enter_sleep();
}
```

## 🎯 应用导向推荐

### 电池供电 (>1 年续航)

**推荐传感器**:
- SHT40 (温湿度)
- LPS22HB (气压)
- SGP40 (VOC, 低频测量)

**策略**:
- 测量频率 <1 次/分钟
- 使用单次测量模式
- MCU 深度睡眠为主

### 无人机/机器人 (实时性优先)

**推荐传感器**:
- BMI088 (IMU, 高 ODR)
- LPS22HB (气压, 中 ODR)
- VL53L1X (ToF, 按需)

**策略**:
- 使用 FIFO 缓冲
- 智能中断唤醒
- 功耗次要，性能优先

### 可穿戴设备 (平衡型)

**推荐传感器**:
- BMI270 (IMU, 低功耗)
- SHT40 (温湿度)
- MAX30102 (心率, 脉冲测量)

**策略**:
- 活动识别唤醒
- 自适应 ODR
- 严格功耗预算
