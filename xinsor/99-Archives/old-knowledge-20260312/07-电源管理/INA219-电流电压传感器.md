# INA219 电流电压传感器

> **一句话描述:** TI 高精度电流/电压/功率监测芯片，I2C 接口，内置 12 位 ADC，支持 0-26V 母线电压

> **状态:** ✅ 完成

## 🎯 概述

**INA219** 是 Texas Instruments 生产的高精度电流/电压/功率监测芯片，集成精密分流放大器、12 位 ADC 和 I2C 接口，可测量母线电压、分流电压 (电流) 并计算功率，广泛应用于电源管理、电池监测、电机控制等场景。

**核心特性:**
- ✨ **三合一测量:** 电压 + 电流 + 功率
- ✨ **宽电压:** 0-26V 母线电压测量
- ✨ **高精度:** 12 位 ADC，0.1% 精度
- ✨ **可编程增益:** 4 档分流电压量程 (±40mV 到 ±320mV)
- ✨ **I2C 接口:** 4 个可选地址
- ✨ **报警功能:** 可编程过流/过压报警

**典型应用:** 电池电量监测、电机驱动电流检测、电源管理、太阳能系统、USB 功率计、智能插座

## 🔌 硬件设计

### 引脚定义

```
        INA219 顶视图 (SOIC-8)
        
    VS+  1 ○ ○ 8 VDD
    IN-  2 ○ ○ 7 SCL
    IN+  3 ○ ○ 6 SDA
    GND  4 ○ ○ 5 A0
```

| 引脚 | 名称 | 类型 | 功能描述 |
|------|------|------|----------|
| 1 | VS+ | P | 母线电压正端 |
| 2 | IN- | I | 分流电阻负端 |
| 3 | IN+ | I | 分流电阻正端 |
| 4 | GND | P | 电源地 |
| 5 | A0 | I | I2C 地址选择 |
| 6 | SDA | I/O | I2C 数据线 |
| 7 | SCL | I | I2C 时钟线 |
| 8 | VDD | P | 电源 (2.7V-5.5V) |

### I2C 地址选择

| A0 连接 | I2C 地址 |
|---------|----------|
| GND | 0x40 |
| VDD | 0x41 |
| SDA | 0x44 |
| SCL | 0x45 |

### 典型电路

```
         负载电源 (0-26V)
               │
               │
              ┌┴┐
              │ │ Rshunt (分流电阻)
              │ │ 0.1Ω/1W
              └┬┘
               │
         ┌─────┴─────┐
         │           │
        IN+         IN-
         │           │
        ┌┴───────────┴┐
        │   INA219    │
        │             │
        │ A0 → GND    │  (地址 0x40)
        │             │
        │ SCL → PB6   │
        │ SDA → PB7   │
        │             │
        │ VDD → 3.3V  │
        │ GND → GND   │
        └─────────────┘
```

**设计要点:**
1. **分流电阻:** 
   - 根据最大电流选择阻值
   - 功率额定值 = I² × R
   - 推荐使用 4 端子开尔文连接
2. **去耦电容:** 1μF + 100nF 并联，靠近 VDD 引脚
3. **I2C 上拉:** 4.7kΩ (3.3V 系统)
4. **PCB 布局:** 
   - 分流电阻走线对称
   - 远离大电流干扰
   - 开尔文连接点尽量靠近电阻

## 💻 软件驱动 (STM32 HAL)

### 寄存器定义

```c
#ifndef __INA219_H
#define __INA219_H

#include "main.h"

// INA219 I2C 地址
#define INA219_ADDR_0   0x40
#define INA219_ADDR_1   0x41
#define INA219_ADDR_2   0x44
#define INA219_ADDR_3   0x45

// 寄存器地址
#define INA219_REG_CONFIG         0x00
#define INA219_REG_SHUNT_VOLTAGE  0x01
#define INA219_REG_BUS_VOLTAGE    0x02
#define INA219_REG_POWER          0x03
#define INA219_REG_CURRENT        0x04
#define INA219_REG_CALIBRATION    0x05

// 增益配置
typedef enum {
    INA219_GAIN_40MV   = 0x00,  // ±40mV
    INA219_GAIN_80MV   = 0x01,  // ±80mV
    INA219_GAIN_160MV  = 0x02,  // ±160mV
    INA219_GAIN_320MV  = 0x03   // ±320mV
} INA219_Gain_t;

// ADC 分辨率
typedef enum {
    INA219_ADC_9BIT   = 0x00,   // 9 位，84μs
    INA219_ADC_10BIT  = 0x01,   // 10 位，148μs
    INA219_ADC_11BIT  = 0x02,   // 11 位，276μs
    INA219_ADC_12BIT  = 0x03    // 12 位，532μs
} INA219_ADC_t;

// 数据输出结构
typedef struct {
    float bus_voltage;    // 母线电压 (V)
    float shunt_voltage;  // 分流电压 (mV)
    float current;        // 电流 (A)
    float power;          // 功率 (W)
    HAL_StatusTypeDef last_status;
} INA219_Data_t;

// 句柄结构
typedef struct {
    I2C_HandleTypeDef *hi2c;
    uint8_t address;
    float shunt_resistor;     // 分流电阻 (Ω)
    float current_lsb;        // 电流 LSB (A)
    float power_lsb;          // 功率 LSB (W)
    uint16_t calibration;     // 校准寄存器值
    INA219_Gain_t gain;
} INA219_Handle_t;

#endif
```

### 初始化函数

```c
/**
 * @brief INA219 初始化
 * @param handle: 句柄
 * @param hi2c: I2C 句柄
 * @param address: I2C 地址
 * @param shunt_resistor: 分流电阻值 (Ω)
 * @param max_expected_current: 预期最大电流 (A)
 * @return HAL_StatusTypeDef
 */
HAL_StatusTypeDef INA219_Init(INA219_Handle_t *handle,
                               I2C_HandleTypeDef *hi2c,
                               uint8_t address,
                               float shunt_resistor,
                               float max_expected_current) {
    uint16_t config = 0;
    
    handle->hi2c = hi2c;
    handle->address = address;
    handle->shunt_resistor = shunt_resistor;
    handle->gain = INA219_GAIN_320MV;
    
    // 1. 计算电流 LSB
    // 电流 LSB = 最大预期电流 / 32768
    handle->current_lsb = max_expected_current / 32768.0f;
    
    // 2. 计算功率 LSB = 电流 LSB × 20
    handle->power_lsb = handle->current_lsb * 20.0f;
    
    // 3. 计算校准寄存器值
    // Calibration = 0.04096 / (电流 LSB × 分流电阻)
    handle->calibration = (uint16_t)(0.04096f / (handle->current_lsb * shunt_resistor));
    
    // 写入校准寄存器
    uint8_t cal_buf[2];
    cal_buf[0] = (handle->calibration >> 8) & 0xFF;
    cal_buf[1] = handle->calibration & 0xFF;
    HAL_I2C_Mem_Write(hi2c, address << 1, INA219_REG_CALIBRATION,
                      I2C_MEMADD_SIZE_16BIT, cal_buf, 2, 100);
    
    // 4. 配置寄存器
    // BIT15: RST = 1 (复位)
    config = (1 << 15);
    HAL_I2C_Mem_Write(hi2c, address << 1, INA219_REG_CONFIG,
                      I2C_MEMADD_SIZE_16BIT, (uint8_t*)&config, 2, 100);
    HAL_Delay(1);
    
    // 5. 配置工作模式
    // 母线电压：12 位，64 次平均
    // 分流电压：12 位，64 次平均
    // 模式：连续测量
    config = (0 << 15) |              // BIT15: RST = 0
             (INA219_GAIN_320MV << 13) |  // BIT14-13: 增益 ±320mV
             (0x07 << 9) |           // BIT11-9: 母线 ADC 12 位，64 次平均
             (0x07 << 6) |           // BIT8-6: 分流 ADC 12 位，64 次平均
             (0x03 << 3) |           // BIT5-3: 模式 连续测量
             (0x00);                 // BIT2-0: 保留
    HAL_I2C_Mem_Write(hi2c, address << 1, INA219_REG_CONFIG,
                      I2C_MEMADD_SIZE_16BIT, (uint8_t*)&config, 2, 100);
    
    HAL_Delay(100);  // 等待稳定
    
    return HAL_OK;
}
```

### 数据读取函数

```c
/**
 * @brief 读取 INA219 数据
 * @param handle: 句柄
 * @param data: 数据输出
 * @return HAL_StatusTypeDef
 */
HAL_StatusTypeDef INA219_ReadData(INA219_Handle_t *handle,
                                   INA219_Data_t *data) {
    uint8_t buffer[2];
    int16_t raw_value;
    
    // 1. 读取母线电压
    data->last_status = HAL_I2C_Mem_Read(handle->hi2c,
                                          handle->address << 1,
                                          INA219_REG_BUS_VOLTAGE,
                                          I2C_MEMADD_SIZE_16BIT,
                                          buffer, 2, 100);
    if (data->last_status != HAL_OK) return data->last_status;
    
    raw_value = (int16_t)((buffer[0] << 8) | buffer[1]);
    // 母线电压 = 原始值 × 4mV
    data->bus_voltage = (raw_value >> 3) * 0.004f;
    
    // 2. 读取分流电压
    data->last_status = HAL_I2C_Mem_Read(handle->hi2c,
                                          handle->address << 1,
                                          INA219_REG_SHUNT_VOLTAGE,
                                          I2C_MEMADD_SIZE_16BIT,
                                          buffer, 2, 100);
    if (data->last_status != HAL_OK) return data->last_status;
    
    raw_value = (int16_t)((buffer[0] << 8) | buffer[1]);
    // 分流电压 = 原始值 × 10μV
    data->shunt_voltage = raw_value * 0.01f;  // mV
    
    // 3. 读取电流
    data->last_status = HAL_I2C_Mem_Read(handle->hi2c,
                                          handle->address << 1,
                                          INA219_REG_CURRENT,
                                          I2C_MEMADD_SIZE_16BIT,
                                          buffer, 2, 100);
    if (data->last_status != HAL_OK) return data->last_status;
    
    raw_value = (int16_t)((buffer[0] << 8) | buffer[1]);
    // 电流 = 原始值 × 电流 LSB
    data->current = raw_value * handle->current_lsb;
    
    // 4. 读取功率
    data->last_status = HAL_I2C_Mem_Read(handle->hi2c,
                                          handle->address << 1,
                                          INA219_REG_POWER,
                                          I2C_MEMADD_SIZE_16BIT,
                                          buffer, 2, 100);
    if (data->last_status != HAL_OK) return data->last_status;
    
    raw_value = (int16_t)((buffer[0] << 8) | buffer[1]);
    // 功率 = 原始值 × 功率 LSB
    data->power = raw_value * handle->power_lsb;
    
    return HAL_OK;
}

/**
 * @brief 仅读取母线电压 (快速)
 * @param handle: 句柄
 * @return 母线电压 (V)
 */
float INA219_ReadBusVoltage(INA219_Handle_t *handle) {
    uint8_t buffer[2];
    int16_t raw_value;
    
    HAL_I2C_Mem_Read(handle->hi2c, handle->address << 1,
                     INA219_REG_BUS_VOLTAGE,
                     I2C_MEMADD_SIZE_16BIT, buffer, 2, 100);
    
    raw_value = (int16_t)((buffer[0] << 8) | buffer[1]);
    return (raw_value >> 3) * 0.004f;
}
```

### 低功耗配置

```c
/**
 * @brief 进入关断模式
 * @param handle: 句柄
 * @return HAL_StatusTypeDef
 */
HAL_StatusTypeDef INA219_Shutdown(INA219_Handle_t *handle) {
    uint16_t config = 0x0000;  // 所有位清零 = 关断
    return HAL_I2C_Mem_Write(handle->hi2c, handle->address << 1,
                              INA219_REG_CONFIG,
                              I2C_MEMADD_SIZE_16BIT,
                              (uint8_t*)&config, 2, 100);
}

/**
 * @brief 单次测量模式
 * @param handle: 句柄
 * @param data: 数据输出
 * @return HAL_StatusTypeDef
 */
HAL_StatusTypeDef INA219_SingleMeasure(INA219_Handle_t *handle,
                                        INA219_Data_t *data) {
    // 配置单次测量 (母线 + 分流)
    uint16_t config = (0 << 15) |
                      (INA219_GAIN_320MV << 13) |
                      (0x03 << 9) |   // 母线 12 位单次
                      (0x03 << 6) |   // 分流 12 位单次
                      (0x01 << 3) |   // 模式：分流单次
                      (0x00);
    
    HAL_I2C_Mem_Write(handle->hi2c, handle->address << 1,
                      INA219_REG_CONFIG,
                      I2C_MEMADD_SIZE_16BIT,
                      (uint8_t*)&config, 2, 100);
    
    // 等待测量完成
    HAL_Delay(1);
    
    // 读取数据
    return INA219_ReadData(handle, data);
}
```

## 🔧 调试技巧

### 常见问题

| 问题 | 解决方案 |
|------|----------|
| I2C 无响应 | 检查 VDD 供电，确认 I2C 地址 |
| 电流读数为 0 | 检查分流电阻连接方向 |
| 电压读数偏差大 | 校准分流电阻值 |
| 功率计算错误 | 确认校准寄存器正确配置 |

### 快速测试

```c
void INA219_QuickTest(INA219_Handle_t *handle) {
    INA219_Data_t data;
    
    printf("=== INA219 Power Monitor ===\r\n");
    
    // 初始化 (0.1Ω分流电阻，最大 2A)
    if (INA219_Init(handle, handle->hi2c, INA219_ADDR_0, 0.1f, 2.0f) != HAL_OK) {
        printf("❌ 初始化失败!\r\n");
        return;
    }
    printf("✅ 初始化成功\r\n");
    printf("电流 LSB: %.6f A\r\n", handle->current_lsb);
    printf("功率 LSB: %.6f W\r\n", handle->power_lsb);
    printf("\r\n");
    
    // 读取 10 次数据
    for (int i = 0; i < 10; i++) {
        if (INA219_ReadData(handle, &data) == HAL_OK) {
            printf("[%d] Vbus: %.2fV | Vshunt: %.2fmV | ",
                   i, data.bus_voltage, data.shunt_voltage);
            printf("I: %.3fA | P: %.3fW\r\n", data.current, data.power);
        }
        HAL_Delay(500);
    }
}
```

### 分流电阻选择

```
分流电阻选择原则:
1. 最大压降 < 增益量程 (通常选 320mV 档)
2. 功率额定值 > I² × R
3. 精度优先选低阻值 (减小功耗)

示例计算:
- 最大电流：2A
- 增益选择：±320mV
- 最大分流压降：320mV
- 分流电阻：R = V/I = 0.32V / 2A = 0.16Ω
- 选标准值：0.1Ω
- 功率额定：P = I² × R = 4 × 0.1 = 0.4W → 选 1W 电阻
```
