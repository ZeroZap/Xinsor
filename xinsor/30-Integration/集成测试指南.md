## 3. 测试用例

### 3.1 功能测试

#### TC-001: 传感器初始化

```javascript
describe('传感器初始化测试', () => {
  it('应成功初始化传感器', async () => {
    const sensor = new SensorAgent(config);
    const result = await sensor.init();
    expect(result).toBe(true);
  });
  
  it('应检测传感器连接状态', async () => {
    const sensor = new SensorAgent(config);
    const connected = await sensor.isConnected();
    expect(connected).toBe(true);
  });
});
```

#### TC-002: 数据读取

```javascript
describe('数据读取测试', () => {
  it('应读取有效数据', async () => {
    const data = await sensor.readData();
    expect(data).toHaveProperty('timestamp');
    expect(data).toHaveProperty('values');
  });
  
  it('数据应在合理范围内', async () => {
    const data = await sensor.readData();
    expect(data.temperature).toBeWithinRange(-40, 85);
  });
});
```

### 3.2 性能测试

#### TC-101: 采样率测试

```javascript
describe('采样率测试', () => {
  it('应达到配置的采样率', async () => {
    const samples = await collectSamples(1000);
    const actualRate = calculateRate(samples);
    expect(actualRate).toBeWithinRange(config.rate * 0.95, config.rate * 1.05);
  });
});
```

#### TC-102: 延迟测试

```javascript
describe('延迟测试', () => {
  it('数据延迟应小于阈值', async () => {
    const latency = await measureLatency();
    expect(latency).toBeLessThan(100); // ms
  });
});
```

### 3.3 稳定性测试

#### TC-201: 长期运行

```javascript
describe('长期稳定性测试', () => {
  it('应稳定运行 24 小时', async () => {
    const result = await runStabilityTest(24 * 60 * 60 * 1000);
    expect(result.errors).toBe(0);
    expect(result.uptime).toBe(24 * 60 * 60 * 1000);
  });
});
```

#### TC-202: 异常恢复

```javascript
describe('异常恢复测试', () => {
  it('应从通信错误中恢复', async () => {
    simulateCommError();
    await waitForRecovery();
    expect(sensor.isConnected()).toBe(true);
  });
});
```

## 4. 测试执行

### 4.1 自动化测试

```bash
# 运行所有测试
npm test

# 运行特定测试
npm test -- --testPathPattern=sensor

# 运行并生成覆盖率报告
npm test -- --coverage
```

### 4.2 手动测试

1. 连接传感器硬件
2. 运行测试脚本
3. 观察输出结果
4. 记录测试数据

### 4.3 测试报告

```bash
# 生成测试报告
npm run test:report

# 导出测试数据
npm run test:export -- --format=json
```
