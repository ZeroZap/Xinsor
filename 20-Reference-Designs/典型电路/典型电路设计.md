## 1. 传感器接口电路

### 1.1 I2C 接口

```
VCC ──┬── 4.7kΩ ── SCL
      │
      └── 4.7kΩ ── SDA
      
MCU ── SCL ──┬── 传感器 SCL
             │
             └── SDA ──┬── 传感器 SDA
                       │
GND ───────────────────┴── 传感器 GND
```

**关键要点**:
- 上拉电阻：4.7kΩ (3.3V 系统)
- 总线电容：< 400pF
- 走线长度：< 30cm
- 靠近传感器放置去耦电容 (0.1μF)

### 1.2 SPI 接口

```
MCU ── SCK ── 传感器 SCK
      ── MOSI ── 传感器 SDI
      ── MISO ── 传感器 SDO
      ── CS ── 传感器 CS
      ── 3.3V ── 传感器 VCC
      ── GND ── 传感器 GND
```

**关键要点**:
- 片选线独立控制
- 时钟极性/相位匹配
- 高速走线等长
- 阻抗控制 (50Ω)

### 1.3 模拟接口

```
传感器 OUT ── RC 滤波 ── MCU ADC
                    │
                   GND
```

**关键要点**:
- RC 滤波：R=100Ω, C=100nF
- 参考电压稳定
- ADC 采样率 > 2 倍信号带宽
- 差分输入优先

## 2. 电源设计

### 2.1 LDO 供电

```
VIN ──┬── C1 10μF ── LDO ── C2 10μF ── VOUT
      │                              │
     GND                            GND
```

**关键要点**:
- 输入输出电容靠近 LDO
- 选择低噪声 LDO
- 注意压差和电流能力

### 2.2 去耦电容

```
VCC ──┬── 10μF (钽电容)
      ├── 100nF (陶瓷)
      └── 10nF (陶瓷) ── 传感器 VCC
```

**关键要点**:
- 多电容并联
- 小电容靠近引脚
- 低 ESL/ESR 电容

## 3. 信号完整性

### 3.1 PCB 布局原则

1. **传感器位置**
   - 远离热源
   - 远离振动源
   - 靠近 MCU (数字传感器)

2. **走线规则**
   - 短而直
   - 避免直角
   - 差分走线等长

3. **地层设计**
   - 完整地平面
   - 避免分割
   - 单点接地 (模拟)

### 3.2 EMI 防护

```
传感器 ── 磁珠 ── 滤波电容 ── MCU
```

**关键要点**:
- 电源入口加磁珠
- 敏感信号加屏蔽
- 外壳接地

## 4. 典型应用电路

### 4.1 温湿度传感器 (AHT20/SHT30)

```
VCC (3.3V) ──┬── 0.1μF ── GND
             │
             └── AHT20 VDD
             
MCU SCL ──────── AHT20 SCL
MCU SDA ──────── AHT20 SDA
GND ──────────── AHT20 GND
```

### 4.2 IMU (MPU6050)

```
VCC (3.3V) ──┬── 0.1μF ── GND
             ├── 10μF ── GND
             │
             └── MPU6050 VDD
             
MCU SCL ──┬── 4.7kΩ ── VCC
          │
          └─── MPU6050 SCL
          
MCU SDA ──┬── 4.7kΩ ── VCC
          │
          └─── MPU6050 SDA
          
MCU INT ──────── MPU6050 INT
GND ──────────── MPU6050 GND
```

### 4.3 气体传感器 (SGP30)

```
VCC (3.3V) ──┬── 0.1μF ── GND
             │
             └── SGP30 VDD
             
MCU SCL ──┬── 4.7kΩ ── VCC
          │
          └─── SGP30 SCL
          
MCU SDA ──┬── 4.7kΩ ── VCC
          │
          └─── SGP30 SDA
          
GND ──────────── SGP30 GND
```
