## 5️⃣ 缺失传感器驱动清单

### 5.1 高优先级 (常用传感器) 🔥

| 传感器 | 型号 | 类型 | 接口 | 厂商 | 应用场景 | 优先级 |
|--------|------|------|------|------|----------|--------|
| **MPU9250** | 9 轴 IMU | 运动 | I2C/SPI | InvenSense | 无人机、机器人 | P0 |
| **LSM6DS3** | 6 轴 IMU | 运动 | I2C/SPI | ST | IoT、可穿戴 | P0 |
| **BMP280** | 气压 | 环境 | I2C/SPI | Bosch | 高度计、气象 | P0 |
| **VL53L1X** | ToF 测距 | 光学 | I2C | ST | 长距离测距 (4m) | P0 |
| **QMC5883L** | 3 轴磁力计 | 磁 | I2C | QST | HMC5883L 替代 | P0 |

**选择理由:**
- MPU9250: MPU6050 升级版，增加磁力计，成本效益高
- LSM6DS3: ST 主流 IMU，与 STM32 生态整合好
- BMP280: BME280 简化版 (无湿度)，成本更低
- VL53L1X: VL53L0X 升级版，量程提升至 4m
- QMC5883L: HMC5883L 停产后的直接替代品

### A. 传感器选型决策树

```
需要测量什么？
│
├─ 运动/姿态
│  ├─ 仅需加速度 → ADXL345
│  ├─ 加速度 + 陀螺仪 (6 轴)
│  │  ├─ 入门/低成本 → MPU6050
│  │  ├─ 高性能/无人机 → BMI088
│  │  └─ 可穿戴 → BMI270
│  ├─ 加速度 + 陀螺仪 + 磁力计 (9 轴)
│  │  ├─ 需要内置融合 → BNO055
│  │  ├─ 高性价比 → MPU9250
│  │  └─ 高性能 → ICM20948
│  └─ 智能 IMU (直接输出姿态)
│     ├─ 通用 → BNO055
│     └─ 低成本 → BNO080
│
├─ 环境参数
│  ├─ 温度 + 湿度
│  │  ├─ 高精度 → SHT30/SHT40
│  │  └─ 低成本 → AHT20/AHT10
│  ├─ 温度 + 湿度 + 气压 → BME280
│  ├─ 仅气压 → BMP280
│  └─ 空气质量 (eCO2/TVOC)
│     ├─ 高精度 → SGP30
│     └─ 低成本 → CCS811
│
├─ 距离/接近
│  ├─ 短距离 (<2m) → VL53L0X
│  ├─ 中距离 (<4m) → VL53L1X
│  └─ 长距离 → TF-Luna (ToF)
│
├─ 磁场/航向
│  ├─ 电子罗盘 → HMC5883L / QMC5883L
│  └─ 9 轴 IMU 内置 → 见运动类
│
└─ 电源监测
   ├─ 电流 + 电压 + 功率 → INA219 / INA226
   └─ 电池电量 → MAX17043
```
