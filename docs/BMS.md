# BMS (电池管理系统) 学习

## 🧠 功能概述

电池管理系统 (Battery Management System, BMS) 是用于监控、管理和保护电池组的智能系统。它确保电池在安全、高效的状态下运行，延长电池寿命，广泛应用于电动汽车、储能系统、电动工具等领域。

### 电压计算

```
标称电压 = S × 3.6V (锂离子)
满电电压 = S × 4.2V
放电截止 = S × 3.0V

示例：13S 电池组
- 标称：13 × 3.6 = 46.8V ≈ 48V
- 满电：13 × 4.2 = 54.6V
- 截止：13 × 3.0 = 39V
```

## 💻 软件算法

### SOC 估算算法

```javascript
// 扩展卡尔曼滤波 (EKF) SOC 估算
class EKFSOCEstimator {
  constructor() {
    this.Q = 0.001;  // 过程噪声协方差
    this.R = 0.01;   // 测量噪声协方差
    this.soc = 0.5;  // 初始 SOC
    this.voltage = 0;
  }

  // 状态更新
  predict(current, dt, capacity) {
    // SOC = SOC_prev - (I * dt) / Capacity
    this.soc = this.soc - (current * dt) / capacity;
    this.soc = Math.max(0, Math.min(1, this.soc));
  }

  // 测量更新
  update(measuredVoltage, ocvCurve) {
    // 从 OCV-SOC 曲线获取估计 SOC
    const estimatedSoc = this.lookupOCV(measuredVoltage, ocvCurve);
    
    // 卡尔曼增益
    const K = this.Q / (this.Q + this.R);
    
    // 修正 SOC
    this.soc = this.soc + K * (estimatedSoc - this.soc);
    
    return this.soc;
  }

  lookupOCV(voltage, ocvCurve) {
    // 查表法获取 OCV 对应的 SOC
    for (let i = 0; i < ocvCurve.length - 1; i++) {
      if (voltage >= ocvCurve[i].v && voltage <= ocvCurve[i+1].v) {
        const ratio = (voltage - ocvCurve[i].v) / 
                      (ocvCurve[i+1].v - ocvCurve[i].v);
        return ocvCurve[i].soc + ratio * (ocvCurve[i+1].soc - ocvCurve[i].soc);
      }
    }
    return 0.5;
  }
}
```
