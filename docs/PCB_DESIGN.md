# PCB 设计实战 (KiCad)

## 🧠 功能概述

PCB (Printed Circuit Board) 是电子产品的骨架，连接所有电子元器件。本学习文档使用 KiCad (开源 EDA 工具) 讲解 PCB 设计全流程，从原理图到 Gerber 输出。

## 🔵 原理图设计

### 1. 项目结构

```
my_project/
├── my_project.kicad_sch      # 原理图文件
├── my_project.kicad_pcb      # PCB 文件
├── my_project.kicad_pro      # 项目配置文件
├── fp-lib-table              # 封装库表
├── sym-lib-table             # 符号库表
└── project/
    ├── project.kicad_sch
    └── project.kicad_pcb
```

### 2. 原理图设计要点

```
┌─────────────────────────────────────────────────────┐
│              原理图设计规范                          │
├─────────────────────────────────────────────────────┤
│                                                     │
│  电源设计：                                         │
│  • 使用不同颜色网络区分 (VCC=红色，GND=绿色)          │
│  • 添加去耦电容 (每 IC 一个 0.1uF)                   │
│  • 电源入口添加 TVS 和保险丝                         │
│                                                     │
│  信号设计：                                         │
│  • 关键信号添加标签 (Label)                         │
│  • 差分对走线标注 (长度匹配要求)                     │
│  • 测试点标注 (TP1, TP2...)                         │
│                                                     │
│  ERC 检查常见问题:                                  │
│  • 未连接引脚 ( floating pins)                      │
│  • 电源冲突 (power conflicts)                       │
│  • 单向引脚连接错误                                 │
│                                                     │
└─────────────────────────────────────────────────────┘
```

## 🟢 PCB 布局布线

### 1. 叠层设计

```
┌─────────────────────────────────────────────────────┐
│              4 层板叠层 (推荐)                        │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │  Top Layer (信号层)                          │   │
│  │  • 关键信号走线                              │   │
│  │  • 元器件布局                                │   │
│  ├─────────────────────────────────────────────┤   │
│  │  GND Plane (完整地平面)                      │   │
│  │  • 参考平面                                  │   │
│  │  • 回流路径                                  │   │
│  ├─────────────────────────────────────────────┤   │
│  │  PWR Plane (电源层)                          │   │
│  │  • 电源分割                                  │   │
│  │  • 大电流走线                                │   │
│  ├─────────────────────────────────────────────┤   │
│  │  Bottom Layer (信号层)                       │   │
│  │  • 一般信号走线                              │   │
│  │  • 背面器件                                  │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  2 层板叠层 (低成本):                               │
│  • Top: 信号 + 电源                                 │
│  • Bottom: 地 (尽量完整)                            │
│                                                     │
│  6 层板叠层 (高速信号):                             │
│  Sig-GND-Sig-PWR-GND-Sig                           │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 2. 布线规则

```
┌─────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────┤
│                                                     │

│  ┌──────────────┬──────────────┬───────────────┐   │
│  │   电流 (A)   │  线宽 (mm)   │  线宽 (mil)   │   │
│  ├──────────────┼──────────────┼───────────────┤   │
│  │     0.1      │     0.2      │       8       │   │
│  │     0.5      │     0.8      │      32       │   │
│  │     1.0      │     1.5      │      60       │   │
│  │     2.0      │     3.0      │     120       │   │
│  │     3.0      │     4.5      │     180       │   │
│  └──────────────┴──────────────┴───────────────┘   │
│                                                     │
│  间距规则：                                         │
│  • 信号 - 信号：6mil (0.15mm) 最小                  │
│  • 电源 - 电源：8mil (0.2mm) 最小                   │
│  • 信号 - 电源：8mil (0.2mm) 最小                   │
│  • 高压 (>50V): 1mm/kV 最小                        │
│                                                     │
│  过孔规则：                                         │
│  • 信号过孔：8/16mil (孔径/焊盘)                   │
│  • 电源过孔：12/24mil                              │
│  • 地过孔：10/20mil                                │
│                                                     │
│  阻抗控制：                                         │
│  • USB D+/D-: 90Ω 差分                             │
│  • 以太网：100Ω 差分                               │
│  • 单端 50Ω (RF 信号)                              │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 3. 布局原则

```
┌─────────────────────────────────────────────────────┐
│              PCB 布局黄金法则                        │
├─────────────────────────────────────────────────────┤
│                                                     │
│  1. 按功能分区：                                    │
│  ┌─────────┬─────────┬─────────┐                   │
│  │  电源区  │  模拟区  │  RF 区   │  ← 敏感区域     │
│  ├─────────┼─────────┼─────────┤                   │
│  │  接口区  │  数字区  │  存储区  │                 │
│  └─────────┴─────────┴─────────┘                   │
│                                                     │
│  2. 去耦电容放置:                                   │
│  • 尽可能靠近 IC 电源引脚                            │
│  • 小电容 (0.1uF) 最靠近引脚                        │
│  • 大电容 (10uF) 可以稍远                          │
│                                                     │
│  3. 晶振布局：                                      │
│  • 靠近 MCU 放置                                    │
│  • 下方完整地平面                                  │
│  • 包地处理，远离干扰源                            │
│                                                     │
│  4. 接口器件布局:                                   │
│  • 靠近连接器放置                                  │
│  • ESD 防护器件靠近接口                            │
│  • 滤波电容靠近器件                                │
│                                                     │
│  5. 热设计：                                        │
│  • 发热器件分散放置                                │
│  • 添加散热过孔                                    │
│  • 大电流走线加宽/开窗                            │
│                                                     │
└─────────────────────────────────────────────────────┘
```

## 🔴 EMI/EMC 设计

### 1. EMI 抑制技术

```
┌─────────────────────────────────────────────────────┐
│              EMI 抑制技术                            │
├─────────────────────────────────────────────────────┤
│                                                     │
│  源头抑制：                                         │
│  • 降低开关速度 (串联电阻)                          │
│  • 扩频时钟 (Spread Spectrum)                       │
│  • 选择低 EMI 器件                                   │
│                                                     │
│  传播路径抑制：                                     │
│  • 完整地平面 (最小回流路径)                        │
│  • 包地处理 (关键信号)                              │
│  • 滤波 (π型/LC 滤波)                               │
│  • 屏蔽 (金属罩)                                    │
│                                                     │
│  敏感电路保护：                                     │
│  • 远离噪声源 (DCDC、晶振)                          │
│  • 增加屏蔽罩                                       │
│  • 使用差分信号                                     │
│                                                     │
└─────────────────────────────────────────────────────┘
```
